Base de Datos para compañía

La base de datos realizada para la empresa de Nutrihana ayuda a guardar información de los clientes, así como de las nutriologas que trabajan con la compañía. Se realizaron trabajos de normalización en la base de datos, llegando hasta la tercera forma normal. Esto se realizó así para que no existieran problemas al momento de generar las queries, ya sea para actualizar o consultar algún dato de la base de datos.
